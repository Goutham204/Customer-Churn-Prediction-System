# Customer Churn Prediction System

Customer Churn Prediction System is a machine learning project designed to identify customers who are likely to discontinue a service. Using the Telco Customer Churn dataset, this project demonstrates a complete ML workflowâ€”from data preprocessing and exploratory analysis to model training, evaluation, and deployment through a user-friendly web interface.

#Project Overview

This project focuses on predicting customer churn using classification algorithms and business-oriented evaluation metrics. The system enables churn probability scoring and customer risk segmentation, helping organizations make proactive, data-driven retention decisions.

#Features

-End-to-End ML Pipeline: Data cleaning, feature engineering, model training, and evaluation

Exploratory Data Analysis (EDA): Identify churn-driving patterns and trends

Classification Models: Train and compare multiple models for churn prediction

Model Evaluation: ROC-AUC, Precision, Recall, F1-score

Risk Segmentation: Assign churn probability scores for customer risk analysis

Web Deployment: Real-time prediction via Streamlit UI and Flask backend
